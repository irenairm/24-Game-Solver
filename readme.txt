Aplikasi Permainan Kartu 24 dengan Algoritma Greedy

back.py:
File ini berisi source code fungsi yang menghasilkan ekspresi dari list yang berisi 4 bilangan bulat [1..13] menggunakan algorithm greedy dan fungsi yang menghasilkan skor dari suatu ekspresi

front1.py:
File ini berisi source code untuk user interface yang menggunakan tkinter

front2.py:
File ini berisi prosedur yang akan memproses file input dengen fungsai greedy24 dan menuliskan hasil tersebut ke file output

main.py:
- Saat dieksekusi, sebuah window akan muncul. Untuk memulai, tekan "CLICK TO PLAY"
- Tombol "Shuffle" mengacak kartu yang ada di dek
- Tombol "Pick" mengeluarkan empat kartu beserta solusi dan skornya
- Tombol "Reset" mengembalikan semua kartu ke dek
- Tombol "Exit" mengeluarkan warning untuk keluar dari permainan
